# Auto Jordi – Demo Website

Premium One-Pager für Auto Jordi (Autogarage, Aarburg).

## Struktur
- `index.html` – Startseite (Hero, Leistungen, Vorher/Nachher, Galerie, Region, Bewertungen, Kontakt)
- `impressum.html` – Impressum
- `datenschutz.html` – Datenschutzerklärung (revDSG)
- `assets/` – Logo & Fotos

## Vor dem Live-Gang
1. **Web3Forms Key**: In `index.html` den Platzhalter `DEIN_WEB3FORMS_KEY` durch den echten Access-Key ersetzen.
2. **noindex entfernen**: Bei eigener Domain die `<meta name="robots" content="noindex, nofollow">` Zeilen entfernen.

Deployment: GitHub → Vercel. Repo: v3dnik/vodnik-demos (Unterordner /auto-jordi).
